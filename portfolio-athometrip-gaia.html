구현 기술

- Next.js, Nest.js, MariaDB, AWS 기반 -
1. 회원가입 기능x, 인증받은 회원들만 접속 가능, Auth.js로 세션관리
2. 데이터 패치(우커머스 데이터베이스에 직접 접근하여 쿼리)
3. webhook, socket.io를 활용한 실시간 주문 데이터 업데이트
4. MSA(백엔드)를 활용하여 회원서비스를 재배포할 때 3004-30004로 전체 프로젝트에 영향 없게 반영
